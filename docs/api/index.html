Employee

Employee-facing clock-in and attendance operations

Record employee clock-in

Records an employee clock-in attempt with geofence and time window validation. Validates location, time window, and determines check status (on_time, late, early_leave).

Authorizations:
FirebaseAuth
Request Body schema: application/json
required
latitude
required
number <double> [ -90 .. 90 ]

GPS latitude coordinate

longitude
required
number <double> [ -180 .. 180 ]

GPS longitude coordinate

isMocked
boolean
Default: false

Whether location is mocked/spoofed

timestamp
string <date-time>

Optional client timestamp (server time preferred)

Responses

Production Firebase Functions endpoint

https://us-central1-{project-id}.cloudfunctions.net/handleClockIn

Request samples

  • Payload
Content type
application/json
{
  • "latitude": 13.756331,
  • "longitude": 100.501762,
  • "isMocked": false,
  • "timestamp": "2019-08-24T14:15:22Z"
}

Response samples

  • 401
  • 403
  • 429
Content type
application/json

  • "success": true,
  • "checkNumber": 1,
  • "status": "on_time",
  • "message": "Check 1 recorded successfully"
}
